
控制反转（Inversion of Control，缩写为IoC），
是面向对象编程中的一种设计原则，可以用来减低计算机代码之间的耦合度。

其中最常见的方式叫做依赖注入（Dependency Injection，简称DI），

还有一种方式叫“依赖查找”（Dependency Lookup）。

通过控制反转，对象在被创建的时候，由一个调控系统内所有对象的外界实体，

将其所依赖的对象的引用传递(注入)给它。--《维基百科》

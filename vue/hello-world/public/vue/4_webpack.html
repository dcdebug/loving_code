Vue 命令行安装

  • 最新版本确认:npm show vue-cli
  • 安装vue: npm install -g vue-cli@version-no
  • 安装版本确认: vue -V|--verison
  • 查看帮助：vue -h

vue 命令默认提供的模版

vue-cli的github地址：https://github.com/vuejs/vue-cli

vue-cli官方提供的模版：https://github.com/vuejs-templates

基于webpack创建自己的项目

  • vue init webpack myproject 初始化我的模版工程
  • cd myproject
  • npm run serve （启动本地的开发环境:localhost:8081的服务启动提示） | npm start ->package.json
